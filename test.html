What is this?

I am trying to test the WMS interfaces for various map services provided by The National Map (TNM). Currently, TNM uses Esri ArcServer 9.1.3, which provides a wealth of options for services. Unfortunately, this wealth of options also provides a wealth of methods to screw things up.

In the table above, I have embedded simple OGC WMS GetMap calls all to the same extent (downtown Denver, straddling the intersection of four 7.5' quads) at about 1:21,000 scale. I have requested the same extent using WGS84 and each of the four variations on Spherical Web Mercator. See this post by Esri Product Specialist, Melita Kennedy for a run-down on Web Mercator EPSG codes in ArcGIS. Here's another great thread from Esri on Spherical Web Mercator.

Theoretically, what you should see is an array of 256x256 map tiles. The first row is the Large Scale Imagery (mostly NAIP but also USGS Urban Areas imagery). The second row is DRGs - scanned USGS Topo Sheets. The third row should be the default rendering of The National Map.
